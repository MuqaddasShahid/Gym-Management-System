# Gym Management System - Windows Forms App

A C# Windows Forms Application built using Visual Studio 2026. This is my first major project to practice .NET concepts.

## 🚀 Application Features
- Dashboard: Central screen with main navigation buttons
- Members Module: Add, Update, Delete, and View Members  
- Trainers Module: Manage trainer details with CRUD operations
- Payments Module: Record member payments 
- Reports Module: Generate and print reports using Report Viewer

## 🛠️ Tech Stack
- Language: C# 
- Framework: .NET Windows Forms
- Database: SQL Server LocalDB
- IDE: Visual Studio 2026

## ▶️ How to Run This Project
1.  Download the GymManagementSystem.zip file from this repo
2.  Extract and open GymManagementSystem.slnx in Visual Studio 2022 
3.  Press F5 to Build & Run the application
